The toolkit we connect throughout the course

You don't need to be a programmer to use any of these — each one is designed for teachers to build and share activities with simple clicks, and each module shows you exactly how.

📈 Desmos

Free graphing calculator and geometry tool. Best for algebra, functions, and anything you want students to manipulate with sliders.

🧩 Mathigon / Polypad

A virtual manipulatives playground — pattern blocks, geoboards, algebra tiles — for concrete, hands-on exploration.

✋ Graspable Math

Lets students drag and rearrange terms inside a real equation, making symbolic algebra feel physical and reversible.

About this site (a note for Dr. Artorn)

This project doubles as your first website. Every module page reuses the same three files: assets/css/style.css (how things look), assets/js/main.js (small shared behaviors), and one index.html per module (the actual content). Once you're comfortable with Module 4 — the fully built example — you can copy its file, rename it, and rewrite the content for Modules 1, 2, 3, and 5 using exactly the same pattern.

See README.md in the project folder for a step-by-step explanation of how the Desmos / Mathigon / Graspable Math embeds work, and how to publish this site to a live link (GitHub Pages) so participants can open it from their own laptops or phones during the training.
