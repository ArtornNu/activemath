📍 1. การแปลงเชิงเส้น (Linear Transformation) [คลิกเพื่อเล่น]

สมการ Ax = b คือการแปลงพิกัดใน Input Space ไปสู่ตำแหน่งเป้าหมายใน Output Space ผ่านการขยาย บิด หรือยืดแกนด้วยเมทริกซ์ A

📐 2. Determinant (det) อัตราส่วนพื้นที่ [คลิกเพื่อเล่น]

det(A) บอกอัตราส่วนขยายพื้นที่ของตารางพิกัด หาก det(A) = 0 พื้นที่จะถูกย่นบีบกลายเป็นเส้นตรงหรือจุด ทำให้ไม่สามารถหา Inverse Matrix ย้อนกลับได้

🗝️ 3. Cramer's Rule อัตราส่วนพื้นที่แก้สมการ [คลิกเพื่อเล่น]

สามารถคำนวณหาค่า x และ y ได้โดยตรงจากอัตราส่วนขยายพื้นที่:
x = det(A_x) / det(A) และ y = det(A_y) / det(A)

🔄 4. Inverse Matrix (A⁻¹) ย้อนกระบวนการ [คลิกเพื่อเล่น]

A⁻¹ = (1 / det(A)) × Adj(A)
• Adj(A) ทำหน้าที่ดึงแกนบิดเอียงให้กลับมาตั้งฉาก
• 1 / det(A) ทำหน้าที่ย่อ/ยืดพื้นที่กลับสู่ 1 ยูนิตมาตรฐานเดิม
🌟 ทำไมต้องมองแบบนี้? มุมมองนี้ทำให้เรา เร็วขึ้น (หาคำตอบได้เลยไม่ต้องแก้สมการทีละขั้น), เห็นภาพชัดขึ้น (รู้ทันทีว่าทำไมบางระบบไม่มีคำตอบ เมื่อ det = 0), และ ต่อยอดได้ไกลขึ้น — แนวคิดเดียวกันนี้ใช้ได้แม้สมการจะมีมากกว่า 2 ตัวแปร และเป็นพื้นฐานของกราฟิกคอมพิวเตอร์กับ AI ด้วย!
✍️ คำถามชวนคิด (สรุปด้วยคำพูดของคุณเอง):

det(A) และ Cramer's Rule ให้คำตอบของระบบสมการได้โดยไม่ต้องแก้สมการทีละขั้นตอนได้อย่างไร? และ Inverse Matrix เกี่ยวข้องกับ "การย้อนกลับ" การแปลงเชิงเส้นในแบบที่คุณเข้าใจอย่างไร? ลองอธิบายด้วยคำพูดของตัวเอง เหมือนกำลังสอนเพื่อนที่ยังไม่เคยทำกิจกรรมนี้
