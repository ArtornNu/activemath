Planned outline

  • A historical hook: problem solving through history — the session slides reference the Code of Hammurabi (r. 1792–1750 BCE) era as an entry point into "mathematics has always been about solving real problems."
  • What counts as a "problem" vs. an "exercise": a task is only a problem if the solver doesn't already know the method.
  • Pólya's four phases (understand, plan, carry out, look back) as a shared vocabulary for the room.
  • A live problem-solving task, attempted first individually, then compared in groups — candidate: an open-ended Desmos or Polypad exploration with no single correct path.
  • Facilitator guide: how to resist giving away the method, and how to run a productive "look back" discussion.
